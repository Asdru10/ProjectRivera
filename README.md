# Transportes y Excavaciones Rivera
This project is a machinery and fleet management system developed for Excavaciones y Transportes Rivera, a company that provides earthmoving, excavation, transportation, and asphalt services.
The system is designed to manage heavy machinery and vehicles, track their usage, register maintenance activities, and support daily operational control. It helps improve asset management, operational efficiency, and decision-making by centralizing key information related to machinery performance and service history.
This project focuses on real-world business needs, including equipment availability, maintenance records, and operational traceability.

## Features
- Machinery and vehicle management
- Maintenance and service records
- Equipment usage tracking
- Operational history and traceability

## Tech Stack
- ASP.NET Core / C#
- SQL Server
- Web-based interface
- Bootstrap
- JavaScript
