# ProjectRivera
# A machinery and fleet management system developed for a construction and transportation company, focused on tracking equipment usage, maintenance, and operational control.

## Features
- Machinery and vehicle management
- Maintenance and service records
- Equipment usage tracking
- Operational history and traceability

## Tech Stack
- ASP.NET Core / C#
- SQL Server
- REST API
- Web-based interface
- Bootstrap
- JavaScript
